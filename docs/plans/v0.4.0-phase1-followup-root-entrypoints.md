# v0.4.0 Phase 1 follow-up — root entry points + missing playwright config

**Branch:** `feat/v0.4.0`
**Repo:** `lib-foundation`
**Builds on:** `0a7b6dd` (Phase 1 import). Source of truth for any re-copied file: lib-acg main `7708ae31b4cd17c1cf81209f07adc88866301b37`.

---

## Why

Verification of `0a7b6dd` found two gaps in the original Phase 1 spec (not Codex errors):

1. **`playwright.config.js` was never imported.** It lives at lib-acg's *root* (not under `playwright/`), so the `playwright/**` import glob missed it. Without it, `npm run test:e2e` (`npx playwright test`) and `tests/acg-restart.spec.js` cannot run. CI stays green because the `acg` job runs Jest (`npm test`), which does not need it — so the break is latent.
2. **`bin/` and `Makefile` belong at the repo root, not buried in the module.** Decision: root-level entry points, module-internal node deps. `bin/acg-*` and a root `Makefile` move/appear at the lib-foundation repo root; `playwright/`, `package.json`, `tests/`, `etc/`, and `node_modules` stay in `scripts/lib/acg/`.

lib-foundation root currently has **no** `bin/` or `Makefile`, so there is nothing to merge.

## Change 1 — add `scripts/lib/acg/playwright.config.js`

Create `scripts/lib/acg/playwright.config.js` with this exact content (verbatim from lib-acg `7708ae31:playwright.config.js`):

```javascript
// @ts-check
const { defineConfig, devices } = require('@playwright/test');

module.exports = defineConfig({
  testDir: './tests',
  testIgnore: ['**/providers/**'],
  timeout: 15000,
  retries: 0,
  reporter: 'list',
  use: {
    headless: true,
  },
  projects: [
    {
      name: 'chromium',
      use: { ...devices['Desktop Chrome'] },
    },
  ],
});
```

`testDir: './tests'` resolves relative to the config file (the module dir); `testIgnore: ['**/providers/**']` skips the Jest unit tests. So `npx playwright test` run from `scripts/lib/acg/` picks up `tests/acg-restart.spec.js` only.

## Change 3 — add root `Makefile`

Create `Makefile` at the repo root. Port from lib-acg `7708ae31:Makefile` with node targets delegating into the module and the `git config core.hooksPath` line dropped (not applicable to lib-foundation).

**Recipe lines MUST be indented with a literal TAB, not spaces** (Make will error otherwise).

```makefile
.PHONY: setup check lint test credential-test restart-test extend-test all help

PROVIDER ?= aws

_ACG_URL := https://app.pluralsight.com/hands-on/playground/cloud-sandboxes
_PROVIDER := $(if $(filter az,$(PROVIDER)),azure,$(PROVIDER))
_ACG_DIR := scripts/lib/acg

help:
	@printf 'ACG module targets (module lives in $(_ACG_DIR)):\n'
	@printf '  setup             — npm install + download Playwright Chromium browser\n'
	@printf '  check             — node --check all module JS files\n'
	@printf '  lint              — shellcheck bin/ entry-point scripts\n'
	@printf '  test              — run fixture-based Playwright tests (no live session needed)\n'
	@printf '  credential-test   — run bin/acg-credential-test against the ACG portal (happy path)\n'
	@printf '                      optional: PROVIDER=aws|gcp|az  (default: aws)\n'
	@printf '  restart-test      — force sandbox restart then re-extract credentials (restart path)\n'
	@printf '                      optional: PROVIDER=aws|gcp|az  (default: aws)\n'
	@printf '  extend-test       — run bin/acg-extend-test against the ACG portal\n'
	@printf '                      optional: PROVIDER=aws|gcp|az  (default: aws)\n'
	@printf '  all               — run check + lint + test + credential-test + restart-test + extend-test\n'
	@printf '                      optional: PROVIDER=aws|gcp|az  (default: aws)\n'

setup:
	cd $(_ACG_DIR) && npm install && npx playwright install chromium

check:
	cd $(_ACG_DIR) && npm run check

test:
	cd $(_ACG_DIR) && npx playwright test --config playwright.config.js

lint:
	shellcheck -S warning bin/acg-credential-test bin/acg-extend-test

credential-test:
	bin/acg-credential-test "$(_ACG_URL)" --provider "$(_PROVIDER)"

restart-test:
	bin/acg-credential-test "$(_ACG_URL)" --provider "$(_PROVIDER)" --force-restart

extend-test:
	bin/acg-extend-test "$(_ACG_URL)" --provider "$(_PROVIDER)"

all: check lint test credential-test restart-test extend-test
```

## Change 4 — fix the `acg` CI job shellcheck path

**Exact old block:**

